# Claude Code — Install Guide

## Quick Install

```bash
git clone https://github.com/mathiaschu/ecommerce-bi.git ~/.claude/skills/ecommerce-bi
cd ~/.claude/skills/ecommerce-bi
pip3 install pandas numpy
```

Restart Claude Code.

- "Run an RFM analysis on `orders.csv`"
